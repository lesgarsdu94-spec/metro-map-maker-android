# metro-map-maker-android

A lightweight Android app built with Jetpack Compose for designing custom metro and transit maps. Features an interactive canvas to dynamically place stations and connect lines.

## Development Roadmap
- [x] **v0.0.1**: Canvas touch tracking and basic station dot rendering.
- [ ] **v0.0.2**: Station naming and data structures.
- [ ] **v0.0.3**: Line drawing and connectivity logic.
- [ ] **v0.0.4**: Basic UI controls (color selection, clear canvas).
- [ ] **v0.0.5**: Refactoring and preparation for public release.

## Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Canvas API)
- **Architecture:** MVVM (State-driven rendering)

## Setup
Import the project into Android Studio (Ladybug or newer) and run it on an emulator or physical device running Android 8.0 (API 26) or above.
